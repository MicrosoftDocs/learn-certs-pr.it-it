# Microsoft Ignite Cloud Skills Challenge 2021: Esame di Certificazione Gratuito


Grazie per aver completato la [Microsoft Ignite Cloud Skills Challenge - Marzo 2021](https://csc.docs.microsoft.com/ignite/registration/March2021). Tutti i partecipanti che hanno completato almeno una sfida hanno diritto ad un esame di certificazione gratuito, soggetto a [Termini e Condizioni](#terms-and-conditions), per contribuire alla promozione delle competenze e dar prova della propria perizia tecnica a datori di lavoro e colleghi. 

## Dettagli

L'esame di certificazione gratuito è disponibile al riscatto a partire dal 7 aprile 2021 ed è valido fino al 30 giugno 2021. Se non prenoti e sostieni il tuo [esame autorizzato](#eligible-exams) entro quel lasso di tempo, non potrai riscattare la tua offerta.

## Come richiedere il tuo esame gratuito

A seguire troverai le istruzioni passo passo per la richiesta del tuo esame gratuito.

1. Seleziona uno qualsiasi degli [esami autorizzati](#eligible-exams) e clicca sul tasto "Programma con Pearson VUE" per avviare il processo di registrazione online.
<br/>
<br/>
![Clicca sul tasto Programma con Pearson VUE per avviare il processo di registrazione online](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-1.png)
<br/>
<br/>

2. Accedi al tuo Microsoft Certification Profile esistente o creane uno usando un account Microsoft.
<br/>
<br/>
![Accedi al profilo di certificazione Microsoft](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-2.png)
<br/>
<br/>

3. Una volta effettuato l'accesso, conferma la veridicità dei dettagli sul tuo Profilo di Certificazione.
<br/>
<br/>
![Conferma la veridicità dei dettagli sul tuo profilo di certificazione](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-3.png)
<br/>
<br/>

4. Clicca sul tasto "Salva & continua" in fondo alla pagina del Profilo di Certificazione per procedere alla pagina "Sconti sugli Esami".
<br/>
<br/>
![Clicca sul tasto Salva & continua in fondo al profilo di certificazione](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-4.png)
<br/>
<br/>

5. Sulla pagina "Sconti sugli Esami", inserisci l'indirizzo email associato al tuo account e l'email che hai usato per registrarti alla Microsoft Ignite Cloud Skills Challenge (e.g. alias@company.com or alias@outlook.com).
<br/>
<br/>
![Sulla pagina Sconti sugli Esami, inserisci l'email utilizzata per registrarti a un evento](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-5.png)
<br/>
<br/>

6. Se hai il diritto a uno sconto su un esame, riceverai una email di verifica all'indirizzo che hai inserito. Per procedere devi inserire il codice ricevuto via mail. Se non ricevi una mail, puoi reinviare il codice. 
<br/>
<br/>
![Inserisci il codice ricevuto via mail per procedere](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-6.png)
<br/>
<br/>

7. Dopo aver inserito il codice corretto, ti verrà richiesto di collegare i tuoi account. DEVI acconsentire a collegare i tuoi account per ricevere uno sconto.
<br/>
<br/>
![DEVI acconsentire a collegare i tuoi account per ricevere uno sconto](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-7.png)
<br/>
<br/>
La prossima volta che eseguirai l'accesso, se hai diritto a qualche esame scontato per uno di questi account, gli sconti appariranno automaticamente. Per rimuovere un account vai alla [Dashboard dei Certificati](https://aka.ms/certdashboard) e clicca su “modifica il tuo profilo di certificazione”.

8. Vedrai uno sconto apparire in cima. Scegli lo sconto che vuoi riscattare e poi continua verso la cassa cliccando sul tasto dell'esame in programma. 
<br/>
<br/>
![Scegli lo sconto che vuoi riscattare](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-8.png)
<br/>
<br/>

9. Segui i passi indicati sulle schermate successive e fornisci le informazioni richieste per programmare il tuo esame. Durante il processo di check-out, verifica che la somma totale ammonti a 0.00$ e invia l'ordine.
<br/>
<br/>
![Segui i passi sulle schermate successive e fornisci le informazioni richieste per programmare l'esame](images/microsoft-ignite-free-certification-exam-offer-march-2021-step-9.png)
<br/>
<br/>

## FAQ

### Sull'offerta

**Q: Quanti esami posso programmare gratuitamente con questa offerta?**

**A:** Con questa offerta puoi riscattare solo un esame di certificazione autorizzato e gratuito, a prescindere dal numero di sfide completate.

**Q: Quali esami di certificazioni sono idonei a questa offerta?**

**A:** Questa offerta include degli esami di certificazione selezionati. Per una lista degli esami di certificazione idonei, consulta la sezione "[Esami idonei](#eligible-exams)”.

**Q: Se mi sono registrato per Microsoft Ignite ma non mi sono iscritto alla Cloud Skills Challenge o per completarla, posso comunque sfruttare questa offerta?**

**A:** No, questa offerta è riservata a coloro che hanno completato una delle sfide di Microsoft Ignite Cloud Skills Challenge – Marzo 2021.

### Programmare il tuo esame gratuito

**Q: Devo sostenere l'esame o il 30 giugno 2021 o prima?**

**A:** Sì, devi sostenere l'esame o il 30 giugno 2021 o prima di quella data.

**Q: Sto riscontrando dei problemi nella programmazione del mio esame gratuito. Dove posso ottenere supporto?**

**A:** Puoi ottenere aiuto contattando [Microsoft Certification Support](https://trainingsupport.microsoft.com/en-us/mcp/forum). Un moderatore risponderà nell'arco di un giorno lavorativo.

**Q: Ho la possibilità di sostenere l'esame programmato di persona o online?**

**A:** Sì, hai la possibilità di sostenere l'esame programmato di persona tramite Pearson Vue in un centro di test locale (se presente) o via [online](/learn/certifications/online-exams). I centri di test potrebbero essere chiusi temporaneamente a causa delle restrizioni imposte dalle misure sanitarie COVID-19. Rimani aggiornato sulle attuali [informazioni per la disponibilità di test da Pearson Vue](https://home.pearsonvue.com/coronavirus-update).

### Politica di Cancellazione e Riprogrammazione

**Q: Posso riprogrammare l'appuntamento per il mio esame di questa offerta se ne avessi la necessità?**

**A:** Sì, puoi riprogrammare l'appuntamento per il tuo esame, ma devi farlo prima della scadenza dell'offerta. Consulta la [Politica di Cancellazione e Riprogrammazione](/learn/certifications/certification-exam-policies#cancellation-and-reschedule-policy) per maggiori informazioni.

**Q: Ci sono dei costi aggiunti quando riprogrammo o cancello l'appuntamento per il mio esame?**

**A:** Se non cancelli con sufficiente anticipo, potresti perdere l'offerta gratuita per l'esame. Se non riprogrammi con sufficiente anticipo, verrà applicata una commissione. Per più informazioni consulta la [Politica di Cancellazione e Riprogrammazione](/learn/certifications/certification-exam-policies#cancellation-and-reschedule-policy).

**Q: Cosa succede se ho mancato l'appuntamento del mio esame senza cancellarlo o riprogrammarlo?**

**A:** Se hai mancato l'appuntamento del tuo esame e non lo hai cancellato o riprogrammato come previsto dalla [Politica di Cancellazione e Riprogrammazione](/learn/certifications/certification-exam-policies#cancellation-and-reschedule-policy) potresti perdere il diritto all'esame gratuito di questa offerta.

### Politica di Ripetizione

**Q: Posso ripetere gratuitamente un esame se non l'ho passato la prima volta sfruttando l'offerta dell'esame di certificazione gratuito?**

**A:** No, l'offerta è valida per un solo tentativo e le ripetizioni dell'esame devono essere programmate separatamente. Per maggiori informazioni sulla ripetizione di esami di certificazione, consulta la nostra [Politica di Ripetizione Esami](/learn/certifications/certification-exam-policies#security-policies).
